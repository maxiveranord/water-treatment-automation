# 💧 Water Treatment Automation System

## 📌 Overview

This project presents an automated water treatment system using PLC and SCADA technologies to monitor and control industrial processes.

---

## 🎯 Objective

* Reduce manual operation
* Improve system efficiency
* Ensure safe plant operation

---

## ⚙️ System Description

The system automates:

* Water level monitoring
* Pump control (ON/OFF)
* Filtration process
* Alarm system

---

## 🧠 Technologies Used

* PLC (Siemens / Allen-Bradley)
* SCADA (WinCC / Ignition)
* Ladder Logic
* HMI
